# VNPT_YOLO_RB3 Hướng dẫn sử dụng

Thực hiện triển khai YOLOv8 trên Qualcomm Dragonwing RB3 Gen2 gồm 3 bước, hiện tại trong repo đã thực hiện xong 3 bước với output, anh có thể thực hiện ở bất cứ bước nào hoặc deploy lên board luôn (cũng là bước 3):

1. Quantize model YOLO sang INT8: anh đọc hướng dẫn ở [quantize.md]

2. Compile app nhận diện: anh đọc hướng dẫn ở [build.md]

3. Deploy app lên board: anh đọc hướng dẫn ở [deploy.md]
